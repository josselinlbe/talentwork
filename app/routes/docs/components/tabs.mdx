```tsx

  <Tabs
    asLinks={false}
    onSelected={(selected) => {
      setSelectedTab(selected);
    }}
    className="w-full sm:w-auto"
    tabs={[
      { name: "Tab 1" },
      { name: "Tab 2" },
      { name: "Tab 3" },
    ]}
  />

    {selectedTab === 0 ? (
      <div>Tab 1 Content...</div>
    ) : selectedTab === 1 ? (
      <div>Tab 2 Content...</div>
    ) : (
      <div>Tab 3 Content...</div>
    )}
  </div>
</div>
```
