# App Portal

## SaaS-ready user pages

- **Dashboard** - Entity usage indicators.
- **Command Palette** - Application commands.
- **Profile** - Users profile settings.
- **Members** - Manage the account (tenant) users.
- **Subscription** - Let your users manage their subscription.
- **Linked Accounts** - Let 2 tenants share an entity row.
- **Account** - Tenant name, slug, and icon.
- **API** - Tenant API Keys and logs.
- **Audit Trails** - Application usage events.
