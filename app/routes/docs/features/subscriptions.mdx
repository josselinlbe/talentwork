# Subscriptions

## Pricing Models

- [Flat-rate](https://stripe.com/docs/products-prices/pricing-models#flat-rate)
- [Per-seat](https://stripe.com/docs/products-prices/pricing-models#per-seat)
- [Usage-based](https://stripe.com/docs/products-prices/pricing-models#usage-based-pricing) ([IN CONSTRUCTION](/docs/roadmap))

## Feature limits

Plan features can be:

- **Monthly** _(e.g. 10 employees/month)_
- **Max** _(e.g. 100 employees)_
- **Not included** _(e.g. priority support)_
- **Included** _(e.g. integrate with Google Sheets)_
- **Unlimited** _(e.g. unlimited employees)_
