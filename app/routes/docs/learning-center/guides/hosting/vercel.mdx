# Vercel

Talentwork hosts on [Vercel](https://vercel.com).

## Prisma Schema Changes

Whenever you modify your Prisma schema, you have two ways of updating Vercel's schema.

1. Add a `vercel-build` [script](https://stackoverflow.com/a/71146297/5697060).
2. Redeploy the latest Vercel Deployment unchecking _"Redeploy with existing Build Cache."_.

Otherwise you could get errors in production like:

<DocTutorialImage
  alt="Schema error"
  src="https://yahooder.sirv.com/saasrock/guides/hosting/vercel/schema-error.png"
/>

## Errors

### 504 Gateway Timeout

If you have this error, make sure that your **Database** provider is on the **same region** as your **Vercel** Deployment.

  alt="504 Gateway Timeout"
