# Publish a Blog Post

## Goals:

- Publish a blog post

## Steps

ðŸ’¿ Go to the `/admin/blog` route.

ðŸ’¿ Click on the _**"Write"**_ button.

ðŸ’¿ Define the **Title**, **Slug**, **Description**, and **Cover image** (if any).

<DocTutorialImage
  alt="Write"
  src="https://yahooder.sirv.com/saasrock/guides/blogging/publish-a-blog-post/write.png"
/>

ðŸ’¿ Set the **Author**, **Category**, **Tags** (separated by commas), **Date**, and Reading time.

<WarningBanner title="WARNING">
  Currently, you can only add <b>Authors</b> and{" "}
  <b>Categories</b> when seeding the database. Subscribe to
  the issue #71 for this.
</WarningBanner>

<DocTutorialImage
  alt="Set Blog Post Details"
  src="https://yahooder.sirv.com/saasrock/guides/blogging/publish-a-blog-post/details.png"
/>

ðŸ’¿ Write some markdown content.

<WarningBanner title="WARNING">
  Currently, you can only copy-paste markdown content,
  there's no content editor, yet. Subscribe to the issue #59
  for this.
</WarningBanner>

If you don't have any markdown content, copy-paste this:

```md
# Title

## Subtitle

Lorem ipsum...

- [Item 1](https://saasrock.com/)
- _Item 2_
- **Item 3**
```

ðŸ’¿ Click on the _**"Preview Markdown"**_ button.

<DocTutorialImage
  alt="Markdown Content"
  src="https://yahooder.sirv.com/saasrock/guides/blogging/publish-a-blog-post/markdown.png"
/>

ðŸ’¿ Save the blog post.

It will redirect you to the blog post URL.

<DocTutorialImage
  alt="Blog Post"
  src="https://yahooder.sirv.com/saasrock/guides/blogging/publish-a-blog-post/post.png"
/>

ðŸ’¿ Test that the blog post is not published. Open an _incognito_ tab, and copy-paste the blog post URL.

Blog post not found:

<DocTutorialImage
  alt="Blog Post Not Found"
