# Plan Entity Limits

## Goals:

- Set up _**"NOT INCLUDED"**_ for the **Free** plan
- Set up a _**"MAX"**_ limit for the **Standard** plan
- Set up a _**"MONTHLY"**_ limit for the **Enterprise** plan

## Steps

ðŸ’¿ Create a Custom Entity _**"Employee"**_.

ðŸ’¿ Add a _**"Name"**_ property.

<DocTutorialImage
  alt="Employee Properties"
  src="https://yahooder.sirv.com/saasrock/guides/entities/shared-entity-rows/properties.png"
/>

ðŸ’¿ Go to [/admin/setup/pricing](/admin/setup/pricing) and click on _**"Click here to create them"**_.

You should have 3 plans:

- **Free**
- **Standard**
- **Enterprise**

<DocTutorialImage
  alt="Plans"
  src="https://yahooder.sirv.com/saasrock/guides/entities/plan-entity-limits/plans.png"
/>

ðŸ’¿ Edit the **Free** plan and add a **Feature** with the following values:

- **Title**: 0 employees
- **Type**: Not included
- **Name**: Employees

<DocTutorialImage
  alt="Plans"
  src="https://yahooder.sirv.com/saasrock/guides/entities/plan-entity-limits/free.png"
/>

ðŸ’¿ Now add the following for the **Standard**:

- **Title**: 1 employee/month
- **Type**: Monthly
- **Name**: Employees
- **Value**: 1

<DocTutorialImage
  alt="Plans"
  src="https://yahooder.sirv.com/saasrock/guides/entities/plan-entity-limits/standard.png"
/>

ðŸ’¿ And for the third plan - **Enterprise** - add:

- **Title**: 2 employees
- **Type**: Max
- **Name**: Employees
- **Value**: 2

<DocTutorialImage
  alt="Plans"
  src="https://yahooder.sirv.com/saasrock/guides/entities/plan-entity-limits/enterprise.png"
/>

You should have 3 plans with the column **Employees**:

<DocTutorialImage
  alt="Plans with Employee Limits"
  src="https://yahooder.sirv.com/saasrock/guides/entities/plan-entity-limits/limits.png"
/>

ðŸ’¿ Go to [/app/acme-corp-1/employees/new](/app/acme-corp-1/employees/new).

Since we've already added the **Employees** feature, and **ACME CORP 1** does not have a subscription, we cannot add any **Employees**.

**Expected Error**: _You don't have an active subscription_.

<DocTutorialImage
  alt="No subscription"
  src="https://yahooder.sirv.com/saasrock/guides/entities/plan-entity-limits/no-subscription.png"
/>

ðŸ’¿ Upgrade to the **Free** plan and try again.

**Expected Error**: _Not included in current plan_.

<DocTutorialImage
  alt="Free Plan"
  src="https://yahooder.sirv.com/saasrock/guides/entities/plan-entity-limits/free-plan.png"
/>

ðŸ’¿ Upgrade to the **Standard** plan and try again.

You should be able to add 1 employee.

ðŸ’¿ Try to add a second employee.

You should not be able to add a third employee since the **Enterprise** plan has a limit of **2 Employees**.

**Expected Error**: _You've reached the limit this month (1) for Employees_.

<DocTutorialImage
  alt="Standard Plan"
  src="https://yahooder.sirv.com/saasrock/guides/entities/plan-entity-limits/standard-plan.png"
/>

ðŸ’¿ Upgrade to the **Enterprise** plan.

ðŸ’¿ Add a second employee.

ðŸ’¿ Add a third employee.

**Expected Error**: _You've reached the limit (2) for Employees_.

<DocTutorialImage
  alt="Enterprise Plan"
  src="https://yahooder.sirv.com/saasrock/guides/entities/plan-entity-limits/enterprise-plan.png"
/>
