# Shared Entity Rows

## Goals:

- Add an **Employee** for 2 tenants

## Steps

ðŸ’¿ Create a Custom Entity _**"Employee"**_, and check **Requires linked accounts**.

<DocTutorialImage
  alt="Created API Key"
  src="https://yahooder.sirv.com/saasrock/guides/entities/shared-entity-rows/employee-entity.png"
/>

ðŸ’¿ Add a _**"Name"**_ property.

<DocTutorialImage
  alt="Employee Properties"
  src="https://yahooder.sirv.com/saasrock/guides/entities/shared-entity-rows/properties.png"
/>

ðŸ’¿ Go to `/admin/accounts` and click on the slug _**"acme-corp-1"**_.

It will redirect you to the _**Acme Corp 1**_ _Dashboard_.

ðŸ’¿ Create a new **Employee**, and select a **Linked Account**:

<DocTutorialImage
  alt="New Employee"
  src="https://yahooder.sirv.com/saasrock/guides/entities/shared-entity-rows/new-employee.png"
/>

ðŸ’¿ Now the **Employee** should be accessible from `/app/acme-corp-1/employees` and `/app/acme-corp-2/employees`:

<DocTutorialImage
  alt="Employee Properties"
  src="https://yahooder.sirv.com/saasrock/guides/entities/shared-entity-rows/tenant-2.png"
/>

<InfoBanner title="NOTE">

  for the <b>Tenant</b> that created it.
