# Create a Custom Entity

## Goals:

- Create a Custom **Entity**
- Add **Properties** of types: _**TEXT**_, _**NUMBER**_, _**DATE**_, _**BOOLEAN**_, _**ENTITY**_, _**MEDIA**_, and _**SELECT**_
- Add a **Entity Rows** with the _Autogenerated Views and Forms_

## Steps

ðŸ’¿ Go to the [/admin/entities](/admin/entities) page.

<DocTutorialImage
  alt="Entities"
  src="https://yahooder.sirv.com/saasrock/guides/entities/create-a-custom-entity/entities.png"
/>

ðŸ’¿ Click on the _**New**_ button.

ðŸ’¿ Create a **Proyect** entity with the following values:

- **Name**: project
- **Slug**: projects
- **Title**: Project
- **Plural**: Projects
- **Prefix**: PRO

ðŸ’¿ Click on the _**Save**_ button.

You'll be redirected to the [/admin/entities/projects/properties](/admin/entities/projects/properties) route.

ðŸ’¿ Add a _**"Name"**_ Property of type _**"Text"**_:

<DocTutorialImage
  alt="Project Properties"
  src="https://yahooder.sirv.com/saasrock/guides/entities/create-a-custom-entity/project-properties.png"
/>

ðŸ’¿ Now let's create the _**Task**_ entity, with the following values:

- **Name**: task
- **Slug**: tasks
- **Title**: Task
- **Plural**: Tasks
- **Prefix**: TSK

ðŸ’¿ Let's grab an icon, go to [icons8](https://icons8.com/icons/set/task), pick an icon, click download, and copy the URL:

You can use this URL: `https://img.icons8.com/color/48/000000/task--v1.png`.

<DocTutorialImage
  alt="Icon"
  src="https://yahooder.sirv.com/saasrock/guides/entities/create-a-custom-entity/icon.png"
/>

ðŸ’¿ Paste the URL in the **Icon** field:

<DocTutorialImage
  alt="Task Entity"
  src="https://yahooder.sirv.com/saasrock/guides/entities/create-a-custom-entity/task.png"
/>

ðŸ’¿ Click on the _**Save**_ button, and add the following properties:
