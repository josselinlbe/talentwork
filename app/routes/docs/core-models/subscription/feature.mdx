- Belongs to a [Product](product).
- Used to limit SaaS users [Custom Entity Rows](/docs/core-models/custom-entity/row) usage based on their [Subscription](/docs/core-models/subscription/product).

## Model

```
model SubscriptionFeature {
  id                    String              @id @default(cuid())
  subscriptionProductId String
  subscriptionProduct   SubscriptionProduct @relation(fields: [subscriptionProductId], references: [id], onDelete: Cascade)
  order                 Int
  title                 String
  name                  String
  type                  Int // 0 = NOT_INCLUDED, 1 = INCLUDED, 2 = MONTHLY, 3 = MAX, 4 = UNLIMITED
  value                 Int
}
```
