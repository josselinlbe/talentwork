- A product can have many [Prices](price), and many [Features](feature).

## Model

```
model SubscriptionProduct {
  id          String                @id @default(cuid())
  stripeId    String
  order       Int
  title       String
  description String
  badge       String
  active      Boolean
  model       Int // 0 = FLAT_RATE, 1 = PER_SEAT, 2 = USAGE_BASED
  public      Boolean
  prices      SubscriptionPrice[]
  features    SubscriptionFeature[]
}
```
