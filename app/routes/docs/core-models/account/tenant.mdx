# Tenant

- Called _Account_ at **_/app_** and _Tenant_ at **_/admin_**.
- Can have zero or many [Users](/docs/core-models/account/user), linked by the [TenantUser](/docs/core-models/account/tenant-user) model.
- Can have 1 [Subscription Product](/docs/core-models/subscription/product), linked by the [Tenant Subscription](/docs/core-models/subscription/tenant-subscription) model.

## Model

```
model Tenant {
  id                       String                 @id @default(cuid())
  createdAt                DateTime               @default(now())
  updatedAt                DateTime               @updatedAt
  slug                     String                 @unique
  name                     String
  icon                     String
  subscriptionId           String?
  subscription             TenantSubscription?
  users                    TenantUser[]
  invitations              TenantUserInvitation[]
  entities                 Row[]
  logs                     Log[]
  apiKeys                  ApiKey[]
  createdLinkedAccounts    LinkedAccount[]        @relation("createdByTenant")
  asProviderLinkedAccounts LinkedAccount[]        @relation("providerTenant")
  asClientLinkedAccounts   LinkedAccount[]        @relation("clientTenant")
}
```
