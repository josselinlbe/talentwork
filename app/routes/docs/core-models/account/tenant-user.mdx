# Tenant User

- Has 1 [User](/docs/core-models/account/user) and 1 [Tenant](/docs/core-models/account/tenant).
- Defines a user membership to a tenant: **_OWNER_**, **_ADMIN_** or **_MEMBER_**.

## Model

```
model TenantUser {
  id        String           @id @default(cuid())
  createdAt DateTime         @default(now())
  tenantId  String
  tenant    Tenant           @relation(fields: [tenantId], references: [id], onDelete: Cascade)
  userId    String
  user      User             @relation(fields: [userId], references: [id], onDelete: Cascade)
  type      Int
  joined    Int
  status    Int
  roles     TenantUserRole[]
}
```
