## Entity Row Features

**Filters**: URL search filters with beautiful UX.

![Row Filters](https://cdn-images-1.medium.com/max/1600/1*CwLPUxEOhfrMeD0aj1BKPQ.png)

**Tags**: Let your users organize and filter their records with colors.

![Row Tags](https://cdn-images-1.medium.com/max/1600/1*ZIfcWhwWB3vg3-6WDi1R9Q.png)

**Tasks**: Let users have a ToDo list for every Entity Row.

![Row Tasks](https://cdn-images-1.medium.com/max/1600/1*DK6rZvDpqOyN2j7y2LCJtg.png)

**Comments**: Users can now leave comments on every row, including some HTML _(with CSS too)_.

![Row Comments](https://cdn-images-1.medium.com/max/1600/1*-rdX6ezs3HYMXlbi2I2huw.png)

**Workflow**: Customize the Entity Workflow States and Transitions _(actionable steps)_.

![Row Workflow](https://cdn-images-1.medium.com/max/1600/1*yHuejpZLl0ybnsBPX3fsYQ.png)

## API limits

If your application is heavily invested in providing an API for your end-users, then you could link their usage with their current subscription.

![API limits](https://cdn-images-1.medium.com/max/1600/1*qrDEUA2E0EbL0Ua8MhJ7ZQ.png)
